Privacy Policy

Effective date: 24 March 2026

Bhajan Book is a free, offline-first app for reading Gujarati bhajans. We respect your privacy and have designed the app so that it does not collect, store, or share any personal data.

No Data Collection

The app does not collect any information about you. There are no accounts, no login, no user profiles, and no analytics or tracking of any kind. We do not use Firebase, Google Analytics, Amplitude, or any similar service.

On-Device Storage

All your data — playlists, preferences, and settings — is stored locally on your device using SQLite and AsyncStorage. This data never leaves your device and is not accessible to us or any third party.

Network Activity

The app makes only two types of network requests:

  • Catalog sync — The app periodically downloads a public SQLite file from a GitHub repository to check for new or updated bhajans. This is a standard HTTPS download. No user data is sent in the request.
  • Error reports and suggestions — When you choose to submit an error report or bhajan suggestion using the in-app forms, the text you write is sent via EmailJS. Only the freeform text you typed and the bhajan number for context are included. No device identifiers, location, or other personal information is attached.

No Advertising

The app does not display advertisements. No ad networks or ad-related tracking are present in the app.

No Third-Party Data Sharing

We do not share any data with third parties because we do not collect any data to share.

Third-Party Services

The app interacts with two external services, both in limited and transparent ways:

  • GitHub — Catalog updates are downloaded from a public GitHub repository. No user data is transmitted. See the GitHub Privacy Statement.
  • EmailJS — Used only when you explicitly submit an error report or suggestion. See the EmailJS Privacy Policy.

Children's Privacy

The app is suitable for all ages. Because we do not collect any data from any user, there are no special concerns regarding children's privacy.

Changes to This Policy

If we make changes to this privacy policy, the updated version will be posted on this page with a revised effective date.

Contact

If you have questions about this privacy policy, please email bhajanbookapp@gmail.com.
