Support

Report an Error

Found a mistake in the lyrics, transliteration, or translation? Use the Report Error button on any bhajan page in the app. This sends us a message with the bhajan details so we can fix it quickly.

Suggest a Bhajan

Know a bhajan that should be in the collection? Use the Suggest Bhajan button in the app's Settings tab. Tell us the name and any details you have — we review every suggestion.

Contact Us

For anything else, email us at bhajanbookapp@gmail.com.
